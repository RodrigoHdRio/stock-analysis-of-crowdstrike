# Stock Analysis of CrowdStrike

## Overview
This project explores historical stock price data for CrowdStrike (CRWD) using Python.  
The analysis focuses on exploratory data analysis and interactive visualizations to understand price behavior over time.

---

## Tools & Libraries
- Python
- pandas
- yfinance
- plotly (express, graph_objects)

---

## Data
Historical stock price data was retrieved programmatically using the `yfinance` library.  
No datasets are stored in this repository.
